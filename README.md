# 🎬 Movie Collection Manager

A terminal-based Movie Collection Manager built in Python as part of my Python learning journey.

## Features

- Add movies
- View all movies
- Count total movies
- Find the highest-rated movie

## Concepts Used

- Python OOP
- Classes and Objects
- Lists
- Functions
- User Input

## How to Run

```bash
movie manager.py
```
